# kalkulator-konversi-cli-kelompok-5
Kalkulator Konversi CLI merupakan aplikasi berbasis Command Line Interface (CLI) yang dirancang untuk melakukan konversi satuan secara cepat dan efisien. Aplikasi ini mendukung beberapa kategori konversi seperti panjang, berat, dan suhu, serta dibangun menggunakan bahasa C dengan fokus pada kesederhanaan, efisiensi, dan kemudahan penggunaan.
